# Web Scraping Mini Project

This project scrapes all books rated **4 and 5 stars** from the website
https://books.toscrape.com/ and extracts their titles and ratings.

The goal of this project is to practice **web scraping fundamentals** using Python,
including HTML parsing, data filtering, and automation.

---

## Features
- Scrapes multiple pages automatically
- Filters books by rating (4⭐ and 5⭐)
- Extracts book titles and ratings
- Clean and readable output

---

## Technologies Used
- Python
- requests
- BeautifulSoup
- Git & GitHub
